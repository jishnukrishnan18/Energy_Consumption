# Energy_Consumption
# Energy Consumption Prediction using Machine Learning

This Machine Learning project focuses on analyzing and predicting daily energy consumption based on weather conditions such as temperature, rainfall, and wind speed. The project aims to discover hidden patterns in energy usage and understand how environmental factors influence power consumption.

The workflow includes data cleaning, preprocessing, feature engineering, Exploratory Data Analysis (EDA), visualization, regression modeling, and predictive analysis. Multiple visualizations were created to identify trends, seasonal variations, peak consumption periods, and relationships between weather variables and energy demand.

Different regression techniques were applied to build predictive models and evaluate performance using metrics such as R² score and RMSE. The project demonstrates how Machine Learning can support smarter energy management and demand forecasting.
